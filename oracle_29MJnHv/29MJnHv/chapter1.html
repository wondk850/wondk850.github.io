모르는 부분을 만나면 이렇게 의미를 찾아라

지시어, 관련어, 자리 파악을 통한 체계적인 의미 추론 방법

1장 학습 목표

이런 상황에서 도움이 됩니다

  • • 모르는 단어나 구문을 만났을 때 당황하는 경우
  • • 사전에만 의존하여 문맥적 의미를 놓치는 경우
  • • 지시어(this, that, it)가 무엇을 가리키는지 헷갈리는 경우
  • • 문장의 구조가 복잡해 보일 때 어디서부터 분석해야 할지 모르는 경우

1장을 마치면 할 수 있게 되는 것

  • 지시어가 가리키는 대상을 정확히 파악
  • 주변 단어들의 관계를 통한 의미 추론
  • 문장 내 요소들의 역할과 관계 파악
  • 논리적 사고를 통한 체계적인 의미 추론
